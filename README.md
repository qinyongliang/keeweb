# KeeWeb for utools

KeePass是一款开源的密码管理工具。而这是一款基于keeweb的utools密码管理插件。

支持快速检索网站密码

可使用 `ctrl + t` 实现用户名和密码的快速输入

更多快捷方式请看设置

> 如果不想每次打开都需要输入密码，可关闭utools的`隐藏插件后完全退出`

[开源代码](https://github.com/qinyongliang/keeweb)
